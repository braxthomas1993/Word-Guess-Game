Number Guessing Game

 

For every round of ten guesses, the code recycles itself to one particular random number. This random number could be any number from 1 to 100. So for you to play this game you have to guess any number between the aforementioned range in ten rounds, the system will tell you if your guess is too high or too low. The game ends when your guess is same as the choosen random number or when you have exceeded your ten rounds. Goodluck!
